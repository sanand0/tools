Export Gemini conversations as Markdown

This page hosts a ready-to-run script that turns a Gemini conversation into Markdown with YAML frontmatter. Copy the script below, paste it into the Gemini browser console, and the conversation lands on your clipboard.

Why no bookmarklet? Gemini blocks external scripts in bookmarklets, so the console paste route is the only reliable way to run the exporter.

How to use it

  1. Open the Gemini conversation you want to export.
  2. Open DevTools, switch to the Console tab, and paste the script.
  3. Press Enter. The Markdown is copied automatically.
  4. Paste into your notes or repository to keep the full thread.
