# Codex Log Converter

Convert a Codex CLI session log (`.jsonl`) into a readable Markdown recap (messages, reasoning blocks, and tool calls).

## How to use

1. Open `codexlog/index.html` in a browser.
2. Drop a Codex session log from `~/.codex/sessions/yyyy/mm/dd/*.jsonl`.
3. Copy the generated Markdown or download it as a `.md` file.
