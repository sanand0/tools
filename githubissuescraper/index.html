Usage

  1. Open a GitHub issue page.
  2. Click the GitHub Scraper bookmarklet.
  3. (Optional) Enter your GitHub Personal Access Token for higher rate limits.
  4. The issue and timeline are copied to your clipboard as Markdown.
  5. Paste it into your notes, editor, or knowledge base.
