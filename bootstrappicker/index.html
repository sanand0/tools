Bootstrap Theme Picker

Preview Bootstrap palettes in-place

Drag the bookmarklet to any site that uses Bootstrap. A modal will let you preview curated color themes or design your own palette on the live page.

How it works

  1. Drag the bookmarklet to your bookmarks bar.
  2. Visit any Bootstrap-powered page and click the bookmark.
  3. Select a preset palette or type custom hex codes for each color.
  4. Bootstrap variables update instantly so you can judge the look in context.
