# Hacker News Thread Markdown

Bookmarklet to copy a Hacker News thread as nested Markdown bullets.

## Usage

1. Open `index.html` in your browser.
2. Drag the **📰 HN Thread → Markdown** button to your bookmarks bar.
3. On any Hacker News discussion page, click the bookmarklet.
4. The thread is copied in the format:

```
- [1] user: message [2024-01-01T12:00Z]
  - [1.1] user: reply [2024-01-01T12:05Z]
```

Story details are included as the first bullet, with all comments nested beneath.
