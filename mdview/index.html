Open selected Markdown, or the whole page text, in a styled popup with syntax highlighting.

Drag this button to your bookmarks bar:

Markdown View

Loading bookmarklet...

Usage

  1. Select Markdown on a page. If nothing is selected, the bookmarklet uses the page's full text.
  2. Click the Markdown View bookmarklet.
  3. A popup opens and renders the text as Markdown.
  4. Fenced code blocks are highlighted automatically.
